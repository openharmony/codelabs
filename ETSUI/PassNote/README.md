
# PassNote 📒🔐  
基于 OpenHarmony 的多功能轻量级密码管理与个人信息管理应用 Demo

## 📌 项目简介

**PassNote** 是一款基于 **OpenHarmony** 平台开发的多功能密码管理应用示例，旨在探索多重认证机制在移动端应用中的设计与实现。  
应用集成了 **数字密码、图形密码、指纹认证** 等多种验证方式，为用户信息安全构筑多层防护体系。

在保障密码安全的同时，PassNote 还融合了 **便签记录** 与 **课表管理** 等实用功能，实现密码管理与日常学习生活场景的高效结合，打造一站式、轻量化的个人信息管理体验。

本项目为功能性 Demo，适用于 OpenHarmony 学习、功能验证及代码贡献场景。

## ✨ 核心功能一览

### 🔐 多重认证体系

* **数字密码**

  * 固定 6 位纯数字
  * 连续输错 **最多 5 次**
* **图形密码**

  * 至少 **4 个点**
  * 数字不可重复
  * 点位范围为 1～9
* **指纹认证**

  * 当前为 **模拟实现**
  * 预留真实生物识别接口扩展能力

### 🔁 密码管理功能

* 修改密码
* 忘记密码（通过其他认证方式进行身份验证）
* 密码输入默认加密显示（`****`）
* 支持点击 👁 图标查看明文

### 📝 便签管理

* 新增 / 删除便签
* 支持关键字搜索历史便签
* 适合记录临时信息、学习笔记等

### 📅 课表管理

* 设置课程名称、上课时间、上课地点
* 自动生成 **可视化课表界面**
* 模拟“加密课表系统”使用场景

---

## 🧭 应用流程概览

1. 启动应用 → **Index 欢迎页**
2. 进入认证方式选择界面
3. 从 **数字 / 图形 / 指纹** 三种方式中选择一种
4. 验证成功后进入 **HomePage 登录成功界面**
5. 可进入：

   * 个人主页（密码管理）
   * 便签管理
   * 课表管理

## 🗂️ 项目结构说明

```text
/common
├── BioAuthConfig.ets     # 指纹认证相关配置（模拟实现）
├── Config.ets            # 全局配置、认证方式与状态管理
├── NoteManager.ets       # 便签数据管理逻辑
├── ScheduleData.ets      # 课表数据结构与管理

/pages
├── FingerprintPage.ets   # 指纹认证界面（模拟）
├── GraphPassPage.ets     # 图形密码认证界面
├── HomePage.ets          # 登录成功后的主界面
├── Index.ets             # 系统欢迎页（初始入口）
├── NumberPassPage.ets    # 数字密码认证界面
├── PasswordEntry.ets     # 三选一认证方式入口界面
├── ProfilePage.ets       # 个人主页（密码管理）
├── SchedulePage.ets      # 课表管理与可视化界面
```

## 📄 页面功能说明

### 🏁 Index.ets — 系统欢迎页

* 应用初始入口
* 展示欢迎信息（如“欢迎使用加密课表系统”）
* 引导用户进入认证流程

---

### 🔑 PasswordEntry.ets — 认证方式选择

* 提供 **数字密码 / 图形密码 / 指纹识别** 三种方式
* 用户选择任一方式进入对应验证页面

---

### 🔢 NumberPassPage.ets — 数字密码验证

* 仅支持 **6 位纯数字**
* 超过 5 次错误将提示失败
* 用于登录、修改密码等场景

---

### 🔷 GraphPassPage.ets — 图形密码验证

* 至少连接 **4 个不同点**
* 点位不可重复
* 用于替代或辅助数字密码验证

---

### 👆 FingerprintPage.ets — 指纹认证

* 当前为 **模拟指纹识别**
* 模拟系统生物认证流程
* 便于后续接入真实 OpenHarmony 生物识别能力

---

### 🏠 HomePage.ets — 登录成功主界面

* 登录成功后的功能入口
* 可跳转至：

  * 个人主页
  * 便签管理
  * 课表管理

---

### 👤 ProfilePage.ets — 个人主页 & 密码管理

* 展示用户名称
* 密码管理功能：

  * 修改密码
  * 忘记密码（通过其他方式验证身份）
* 输入密码默认加密显示（`****`）
* 支持点击 👁 查看真实内容
* 多重验证确保修改安全性

---

### 📝 便签功能（NoteManager.ets）

* 新增便签
* 删除便签
* 搜索历史便签内容
* 数据集中管理，便于扩展

---

### 📚 SchedulePage.ets — 课表管理

* 设置课程时间、课程地点
* 结构化存储课程数据
* 自动生成 **可视化课表界面**
* 模拟学习场景下的信息加密管理

---

## 🛠️ 技术特点

* 基于 **OpenHarmony ArkUI（ETS）**
* 模块化设计，逻辑与 UI 分离
* 多认证方式统一管理

* 便于二次开发与功能扩展
