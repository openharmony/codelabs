# LifeHub - 生活中心应用

## 项目简介

LifeHub 是一个综合性的个人生活管理应用，融合了健康记录、旅行规划、目标管理、心情日记、技能学习6大核心模块，以及数据统计、搜索、成就、报告、设置、提醒、备份、标签、筛选、帮助等10大辅助功能，为用户提供一站式的数字化生活管理解决方案。

## 核心功能

### 🏠 仪表盘（Dashboard）
- 今日概览：各模块关键数据一目了然
- 快速入口：快速访问常用功能
- 数据卡片：可视化展示统计数据

### 💚 健康模块（Health）
- **健康数据记录**：体重、血压、血糖、运动、睡眠
- **数据可视化**：趋势图表展示
- **健康目标**：设置并追踪健康目标
- **用药提醒**：定时提醒功能

### ✈️ 旅行模块（Travel）
- **行程规划**：创建和管理旅行计划
- **地点收藏**：收藏景点、餐厅、酒店
- **费用管理**：预算和实际花费追踪
- **旅行日记**：记录旅行中的美好时刻

### 🎯 目标模块（Goals）
- **目标设定**：创建长期/短期目标
- **任务分解**：将目标分解为可执行的任务
- **进度追踪**：实时查看目标完成进度
- **目标复盘**：总结完成情况

### 😊 心情模块（Mood）
- **心情记录**：记录每日心情和评分
- **日记功能**：文字、照片、位置、天气
- **心情分析**：心情趋势和日历视图
- **标签系统**：分类管理日记

### 📚 学习模块（Learning）
- **技能管理**：添加和管理学习技能
- **学习记录**：记录学习时长和内容
- **进度追踪**：技能等级和熟练度
- **学习统计**：学习时长和技能分布

### 📝 日记模块（Diary）
- **日记记录**：记录日常生活和想法
- **心情标记**：为日记添加心情标签
- **标签分类**：使用标签管理日记
- **日期筛选**：按日期查看历史日记

## 技术架构

### 数据层
- **统一数据管理**：`LifeHubData` 类管理所有模块数据
- **关系型数据库**：使用 HarmonyOS RDB 存储
- **数据关联**：支持跨模块数据关联

### 视图层
- **模块化设计**：每个模块独立页面
- **统一UI风格**：一致的设计语言
- **响应式布局**：适配不同屏幕

### 核心特性
- 本地数据存储，保护隐私
- 数据可视化展示
- 统一搜索功能
- 跨模块数据统计

4. **数据筛选**：在Dashboard的"工具功能"中点击"数据筛选"筛选和排序数据

## 常见问题

### Q: 如何备份数据？
A: 在Dashboard的"工具功能"中点击"数据备份"，然后点击"开始备份"按钮，复制生成的JSON数据保存即可。

### Q: 如何恢复数据？
A: 在"数据备份"页面，将之前备份的JSON数据粘贴到恢复输入框，点击"开始恢复"即可。

### Q: 如何设置提醒？
A: 在Dashboard的"工具功能"中点击"提醒管理"，然后点击右上角"+"按钮创建新提醒。

### Q: 如何搜索数据？
A: 在Dashboard点击"统一搜索"，输入关键词即可搜索所有模块的数据。

### Q: 如何查看统计数据？
A: 在Dashboard点击"数据统计"，可以查看各模块的详细统计和可视化图表。
